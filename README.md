🍋 Lemonade Land — Fun & Classy Kids Webpage

Welcome to Lemonade Land, a colorful, eye-catching, and kid-friendly webpage made using HTML, CSS, and JavaScript.
This project is designed to be simple, fun, and beautiful, with playful colors, cartoon vibes, animations, and a premium polished footer.

🌟 Features
🍋 Lemonade Hero Section

A bright, bubbly header with animated elements to attract kids instantly.

🍭 Fun & Cute Color Theme

A mix of yellow, pink, mint, and sky-blue to make the page visually delightful.

🧃 Lemonade Menu

A section showcasing different lemonade flavors like:

Classic Lemonade

Strawberry Lemonade

Rainbow Lemonade

Blueberry Splash

Fizzy Lemon Pop

🎉 Fun Facts Section

Cute and interesting lemon facts to keep kids engaged.

🎨 Fun Extras

Colorful emojis and bonus items to make the page lively.

🌈 Smooth Hover Effects

Buttons and cards come alive with soft animations.

⭐ Premium Polished Footer

A modern, clean, golden-highlighted footer giving a classy finish.
